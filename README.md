# QuickPick

**Creating an platform where user can use AI for generating images, videos, answers, music, code.**

#### To run the project - 
* `git clone https://github.com/Vaibhavpratapsingh22/quick-pick.git`
* `git pull`
* `npm run dev`
* `localhost:3000`
